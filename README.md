# flask-base-structure
This is basic structure for more easier to start Flask project
Include follow feature:
<ul>
  <li>Log</li>
  <li>Cache</li>
  <li>DB</li>
  <li>.env</li>
  <li>Csrf</li>
</ul>

*Flask default is single threat, need to start the service with 'gunicorn + gevent' with multi worker
